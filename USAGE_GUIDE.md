## ðŸ”§ Troubleshooting

### Issue: "No active API key found"

**Solution**:
1. Go to Settings > API Keys
2. Make sure you have added a key
3. Check that it's toggled to "Active"
4. Try toggling off and on again

### Issue: "Failed to generate article"

**Possible Causes**:
1. **API key invalid**: Generate a new key
2. **Rate limit exceeded**: Wait 60 seconds and try again
3. **Network error**: Check internet connection
4. **Topic too vague**: Be more specific

### Issue: WordPress publishing failed

**Solutions**:
1. **Test connection** in Settings
2. Check WordPress URL is correct (include https://)
3. Verify application password (no typos)
4. Make sure WordPress REST API is enabled
5. Check if your host blocks API requests

### Issue: Competitor analysis not working

**Solutions**:
1. Check URLs are publicly accessible
2. Some sites block scraping - try different URLs
3. Use recent articles (better success rate)
4. Try 3 URLs instead of 5

### Issue: Internal links not appearing

**Solutions**:
1. Import sitemap first in Settings
2. Make sure you entered a target keyword
3. Check if internal links toggle is enabled
4. Verify imported links are relevant to topic
